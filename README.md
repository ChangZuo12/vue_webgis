##  vue + node + WebGIS项目说明：

### 实践目的：基于VUE框架使用模块化编程方式实现WebGIS的开发

### 主要功能
1. 前后端项目搭建
2. 登录注册（token验证功能）
3. 丰富的地图开发api DEMO集成，地图可视化实现

### 主要技术

> 地图框架：openlayers 5.x / arcgis api / mapbox

> 构建接口文档：Node + express + jwt;

> 构建前端页面：VueCli 3.0 + ElementUI

> 数据请求及拦截：Axios + MongoDB Compass
